# 2. Machine Learning

# 15. Kod (przykład)

```python
from openai import AzureOpenAI

client = AzureOpenAI()

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[
        {
            "role": "user",
            "content": "Hello!"
        }
    ]
)

print(response.choices[0].message.content)
```
